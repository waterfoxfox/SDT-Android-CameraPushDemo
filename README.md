# DEMO说明
本DEMO用于演示Android端 摄像头麦克风采集、编码、调用SDT API发往服务器指定位置。<br>

DEMO可动态切换前后置摄像头、视频软编码硬编码可选。

可通过DEMO设置界面配置服务器IP、房间号，编码分辨率、码率、帧率、FEC相关参数等。
